# inyectoras-arg
Web institucional para Inyectoras Argentinas (inyección de plásticos y servicio técnico). Hecha con Vite + React + TypeScript + Tailwind. Incluye preloader, navegación responsive, hero con stats, showcase de servicios, carrusel de proceso, sección "Nosotros", testimonios y formulario de contacto vía Formspree.
